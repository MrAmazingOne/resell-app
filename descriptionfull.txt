Vision + AI Two-Stage Identification System
Stage 1: iOS Vision Framework (Pre-Processing)

Performs on-device object detection and categorization

Extracts visual features and bounding regions

Supports subject isolation and multi-item extraction

Reduces backend processing time

Prevents render request timeouts (30s execution / 50s spin-down)

Stage 2: Backend AI Agent

The AI agent receives:

One or more cropped images per item

Vision framework metadata

Optional user-provided text

It performs:

Item identification (brand, model, year, category)

Market matching

Valuation analysis

Resale potential evaluation

Backend Processing & Market Intelligence

The backend asynchronously:

Queries eBay API for:

Recent sold listings

Active listings

Price distributions

Market liquidity

Calculates:

Average sale price

Lowest and highest historical prices

Estimated resale price

Cost vs. profit opportunity

Demand and trend signals

The frontend polls for completion to avoid timeout failures.
